# Cognitive Mapping — Doctrinal View

This document explains how classical cognitive concepts
(STM, WM, LTM) map onto the Brain model.

It clarifies what the system remembers, what it forgets,
and why **Task is explicitly forbidden in institutional memory**.

---

## 1. Purpose of This Mapping

Human cognitive science uses terms like:

- Short-Term Memory (STM)
- Working Memory (WM)
- Long-Term Memory (LTM)

These concepts are useful for orientation,
but they are **not** Brain objects.

Brain does not implement human cognition.
Brain implements institutional epistemology.

This mapping exists to prevent confusion,
not to introduce new structures.

---

## 2. Short-Term Memory (STM)

**In human cognition**:
Brief retention of input before processing.

**In Brain**:
STM maps to **Perceptual Capability** + optional **Percept Store**.

Characteristics:

- Raw input
- No meaning assigned
- No truth status
- No responsibility
- Ephemeral by default

Canon permits temporary retention of percepts
without epistemic commitment.

STM is a buffer of the world, not a memory of the system.

---

## 3. Working Memory (WM)

**In human cognition**:
Active mental context during reasoning.

**In Brain**:
WM maps to **ephemeral Cognitive runtime context**.

WM consists of:

- Active Intent
- Relevant Knowledge fragments (read-only)
- Relevant IEM fragments (read-only)
- Current percepts

**Critical rule**: WM is never persisted.

WM exists only while a cognitive flow is active.
When the flow ends, WM disappears.

Canon explicitly states:

> Working context is ephemeral and is not subject to memory.

This is a prohibition, not a limitation.

---

## 4. Long-Term Memory (LTM)

**In human cognition**:
Persistent storage of knowledge and experience.

**In Brain**:
LTM maps **exclusively** to **IEM** (Institutional Epistemic Memory).

If something is not in IEM, it is not LTM.
Regardless of how long it "exists" elsewhere.

Conversely:
Everything in IEM is LTM,
but only because it carries institutional responsibility.

LTM is not storage duration.
LTM is accountability over time.

---

## 5. The Task Prohibition

**Task** is a common operational concept:

- Has identity
- Has status
- Persists over time
- Represents "work in progress"

This makes Task feel like it belongs in memory.

**This is an illusion.**

Task is:

- Not a claim about the world
- Not institutional responsibility
- Not epistemic content
- Not truth

Task is an **orchestration artifact** —
a human and operational convenience.

**Canonical rule**:

> Task is forbidden in IEM.

If Task enters IEM:

- It gains false epistemic weight
- It becomes confused with claims
- Work is remembered instead of responsibility

This was the precise failure mode of earlier models (BOM/IMM).

---

## 6. Where Task Belongs

Task belongs in:

- Human tools (calendars, project managers)
- Operational orchestration layers
- UI and coordination surfaces

Task does not belong in:

- Brain
- IEM
- Any canonical memory structure

Systems may use Task externally.
Brain does not see Task.

---

## 7. Canonical Mapping Table

| Cognitive Concept | Brain Mapping | Canonical Status |
|-------------------|---------------|------------------|
| STM | Perceptual Capability / Percept Store | Permitted, ephemeral |
| WM | Cognitive runtime context | Explicitly ephemeral, never persisted |
| LTM | IEM only | Sole canonical long-term memory |
| Task | None (external only) | Forbidden in IEM |

---

## 8. Why This Matters

If Brain were to model human cognition directly:

- Speculation would become memory
- Work would become truth
- Processes would become claims

Brain models institutional epistemology.
Institutions remember what they are responsible for.
Everything else must be allowed to disappear.

---

## 9. The BOM/IMM Lesson

Earlier models (BOM, IMM) attempted to include Task in memory.

The failure was predictable:

- Task had identity, so it felt persistent
- Task had status, so it felt like state
- Task lasted over time, so it felt like memory

But Task is not about the world.
Task is about work.

And work is not remembered.
Only its consequences are.

---

## Closing Note

Brain remembers only what the institution stands behind.

Everything else — percepts, reasoning, tasks, processes —
exists only long enough to serve its purpose.

Then it disappears.

This is not a loss.
This is discipline.
