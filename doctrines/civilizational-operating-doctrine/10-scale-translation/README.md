# Scale Translation

Civilizational mechanisms do not transfer cleanly across scale.

What works locally can fail nationally.

What works nationally can fail globally.

What works in one culture can become humiliation in another.

This section exists to prevent translation drift.

Translation drift is one of the main ways well-intended coordination becomes coercion.

---

## The core stance

The stance of this section is:

**Translate mechanisms, not narratives.**

Narratives are culture-bound.

Mechanisms can be inspected.

Mechanisms can be adapted.

---

## The civilizational hazard

The hazard is scale translation being done through:

- universalist ideology
- moral superiority
- and humiliation framing

Humiliation produces backlash.

Backlash produces identity hardening.

Identity hardening produces conflict.

Conflict produces coercive closure.

---

## What this section covers

This section defines:

- how local coordination scales nationally
- how national coordination interacts globally
- how to translate across cultures without humiliation
- how to scale without ideology
- how governance of change can evolve doctrine without drift

This is not diplomacy.

It is translation discipline for correction-capable order.

---

## Minimal implementation

A correction-capable order must preserve:

1. translation that avoids humiliation
2. mechanism-first transfer (not slogans)
3. bounded adaptation rather than universal replacement
4. feedback loops that detect translation failure

If translation becomes ideology, it becomes capture.
