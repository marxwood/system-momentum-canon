# Sacred Values

Every civilization has sacred values.

Sacred values are not ideology.

Sacred values are constraints.

They define what cannot be traded for outcomes.

A society without sacred values is not rational.

It is capturable.

Because everything becomes negotiable under pressure.

---

## Sacred values stabilize order

Sacred values stabilize order by providing:

- non-negotiable boundaries
- legitimacy limits
- and refusal rights

These limits prevent legitimacy from becoming a license.

They prevent emergency logic from erasing constraints.

---

## The civilizational hazard

The hazard is sacred values being weaponized.

Weaponized sacred values become:

- purity tests
- enemy classifiers
- justification engines for coercion
- and contradiction killers

Sacred values stabilize when they bound power.

Sacred values destroy correction when they become identity weapons.

---

## Sacred values as constraints vs sacred values as weapons

This module makes one distinction.

### Sacred constraints

Sacred values that:

- apply symmetrically
- bound action
- and preserve contradiction capacity

These stabilize correction-capable order.

### Sacred weapons

Sacred values that:

- apply selectively
- justify humiliation
- and erase contradiction

These accelerate capture.

---

## Failure modes

### Sacred monopoly

One sacred value becomes the only permitted value.

Contradiction becomes treason.

### Sacred inflation

More and more values become sacred.

Sacred inflation makes negotiation impossible.

Negotiation collapse makes coercion attractive.

### Sacred enemy

Sacred values require enemies.

Enemies stabilize identity.

Enemies also stabilize coercion.

### Sacred immunity

Certain actors become sacred.

Sacred actors become immune.

Immune actors become power.

Power becomes capture.

---

## Minimal implementation

A correction-capable order must preserve:

1. at least one sacred constraint that bounds coercion
2. symmetry (sacred values apply to friends and enemies)
3. contradiction protection around sacred claims

If sacred values cannot be contradicted safely, they are no longer constraints.

They are weapons.
