# Correction-Capable Order

This doctrine branch does not treat order as peace.

It treats order as a coordination outcome.

The target is not harmony.

The target is an order that preserves correction.

---

## The target order type

The target order is defined by three properties:

1. **Order that preserves correction**
2. **Order that prevents capture**
3. **Order that minimizes coercion**

These are not moral ideals.

They are operational requirements.

---

## 1) Order that preserves correction

Correction preservation means:

- evidence primitives remain inspectable
- provenance remains contestable
- contradiction remains survivable
- uncertainty remains permitted
- belief update remains possible without identity death

A society can be stable without these.

But it will not be able to repair itself.

---

## 2) Order that prevents capture

Capture prevention means:

- power maps remain visible
- dependence maps remain visible
- constraints survive emergency framing
- oversight is distributed
- accountability is ledgered and bounded
- elites cannot become immune

Capture is not eliminated.

It is made reversible.

---

## 3) Order that minimizes coercion

Coercion minimization means:

- exit paths exist
- safety nets reduce fear
- punishment is predictable and symmetric
- humiliation is not enforcement
- de-escalation is legitimate
- continuity is possible under pressure

Coercion will exist.

The goal is to prevent coercion from becoming the default stabilizer.

---

## The civilizational hazard

The hazard is that societies mistake stability for success.

Stability can be produced by:

- fear
- dependence
- propaganda
- and silence

That is order.

It is not correction-capable.

Correction-capable order is rarer.

It is also the only order that can survive long time horizons without collapse.

---

## Minimal implementation

A correction-capable order must preserve:

1. shared reality anchors
2. legitimacy as bounded permission
3. constraints that survive pressure
4. economics that preserve exit
5. narrative systems that do not destroy evidence
6. survival mechanisms under coercion
7. coordination mechanisms that do not require ideological unity

If these survive, correction survives.

If correction survives, the civilization can repair.
