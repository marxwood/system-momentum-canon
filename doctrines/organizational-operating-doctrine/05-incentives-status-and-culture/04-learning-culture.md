# Learning Culture

Learning culture is not curiosity.

Learning culture is repair discipline.

Organizations that cannot repair cannot learn.

Organizations that cannot learn cannot survive.

---

## Learning is not training

Learning is not courses.

Learning is the system that converts:

- failure
  into
- improved constraints

Without this conversion, failure repeats.

Repeated failure becomes coercion.

Because reality forces correction.

---

## The organizational hazard

The hazard is embarrassment avoidance.

Embarrassment avoidance produces:

- cover-ups
- narrative rewriting
- and scapegoating

All three destroy learning.

---

## Failure modes

### Postmortem theater

Postmortems exist but produce no change.

This is ritual.

### Hidden incidents

Failures are suppressed.

Memory collapses.

### No repair budget

Repair work is deprioritized.

Debt accumulates.

### Punitive learning

Learning is tied to humiliation.

People hide evidence.

---

## Minimal implementation

A correction-capable organization must preserve:

1. post-incident repair obligations
2. memory that survives embarrassment
3. repair capacity protected from urgency
4. incentives that reward learning outcomes

If learning cannot produce repair, the organization will repeat failures until collapse.
