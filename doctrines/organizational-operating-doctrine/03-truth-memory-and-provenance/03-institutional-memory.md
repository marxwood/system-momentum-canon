# Institutional Memory

Organizations do not forget accidentally.

They forget by design.

Forgetting is often rewarded because it removes responsibility.

A correction-capable organization treats memory as:

- continuity infrastructure
- and anti-capture defense

---

## Memory is not documentation

Documentation is not memory.

Memory is what preserves:

- decisions
- their evidence
- their consequences
- and their repairs

Without this, decisions can be rewritten.

Rewriteable history is capture.

---

## The organizational hazard

The hazard is memory collapse.

Memory collapses when:

- ownership is personal rather than institutional
- knowledge is trapped in individuals
- or turnover is used as erasure

Memory collapse produces:

- repeated failure
- permanent reinvention
- and narrative rewriting

---

## Failure modes

### Hero memory

One person remembers everything.

This becomes continuity monopoly.

### Tool memory

Everything is stored, nothing is retrievable.

This becomes amnesia.

### Memory as shame

Past failures are hidden.

This becomes delusion.

### Memory as ideology

A preferred story is preserved.

Contradictions are removed.

---

## Minimal implementation

A correction-capable organization must preserve:

1. decision continuity across turnover
2. provenance bindings for key artifacts
3. repair history, not only outcomes
4. redundancy of memory carriers

If memory cannot persist, accountability cannot persist.

If accountability cannot persist, capture becomes stable.
