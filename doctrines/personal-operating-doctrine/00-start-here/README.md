# Start Here (Personal)

This branch defines how a single actor operates while augmented by an agentic layer.

It is not a philosophy.

It is not a self-improvement program.

It is an operating doctrine: how to preserve correction, dignity, and coherence while living inside real constraints.

---

## What this branch is for

A single actor today does not operate alone.

Even without “AI”, modern life already includes:

- external memory (notes, search, feeds)
- external coordination (calendar, messaging, platforms)
- external incentives (status, reputation, metrics)
- external narrative pressure (media, identity systems)

An agentic layer makes these forces stronger.

This doctrine treats the actor + augmentation as one operating system.

---

## The core problem

The core problem is not productivity.

The core problem is drift.

Drift is when:

- intent becomes implicit
- interpretation becomes reality
- and reaction becomes action

Drift produces:

- self-capture
- dependency
- and quiet coercion

Most “personal systems” fail because they optimize for output while losing correction.

---

## The target outcome

The target outcome is not happiness.

The target outcome is a stable form of order that preserves:

- reality contact
- bounded intent
- repair capacity
- and non-reactive coordination

This is what makes life survivable without becoming a narrative machine.

---

## Vocabulary (minimal)

### Projection

A projection is an operational representation of reality.

A projection is useful.

A projection is never truth.

Examples include:

- calendars
- task lists
- project boards
- financial summaries
- relationship maps

A projection is allowed to be wrong.

It must be correctable.

---

### Surface

A surface is where commitments and actions are expressed.

Surfaces are not metaphysical.

They are interfaces.

Examples include:

- calendar events
- written commitments
- explicit “no”
- published decisions
- messages sent to others

Surfaces create consequences.

---

### Render

A render is a view of a projection or a surface.

Renders are made for readability.

They are not the system itself.

A render can be beautiful and still misleading.

---

## Non-goals

This branch is not:

- spiritual teaching
- therapy
- ideology
- political guidance
- a productivity hack catalog
- a personality system

It is also not a claim that the actor can “control life”.

The doctrine assumes constraint, uncertainty, and conflict.

---

## How to read this branch

The structure is deliberately mechanical:

- posture before technique
- reality contact before plans
- intent before commitments
- repair before optimization

Each section contains:

- a narrative doctrine (how the mechanism works)
- and bounded operational modules (what it implies)

---

## The personal hazard

The hazard is self-capture.

Self-capture occurs when:

- identity becomes a prison
- narratives become mandatory
- and the agentic layer becomes authority

When this happens, the actor loses correction.

The system becomes efficient but unfree.

---

## Minimal implementation

A correction-capable personal operating system preserves:

1. explicit intent and bounded commitments
2. projections treated as correctable representations
3. repair capacity protected from shame and panic
4. augmentation treated as an instrument, never as authority

If these are not preserved, personal “optimization” becomes quiet coercion.
