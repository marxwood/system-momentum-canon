# Provenance and Memory

Personal memory is not a recording.

It is reconstruction.

Reconstruction is necessary.

Reconstruction is also dangerous.

This module treats memory as fallible and therefore subject to provenance discipline.

---

## Memory is a narrative engine

Memory tends to:

- compress
- simplify
- and justify

Under shame or conflict, memory becomes defense.

Defense produces confabulation.

Confabulation feels like truth.

It is not.

---

## Provenance is the antidote

Provenance means:

- where a belief came from
- what evidence it rests on
- and how stable it is

Without provenance, beliefs become floating objects.

Floating beliefs are easily weaponized by emotion.

---

## External memory is not optional

A single actor cannot preserve correction with internal memory alone.

External memory can be:

- notes
- messages
- logs
- or explicit decisions

External memory is not about productivity.

It is about preventing self-deception.

---

## The personal hazard

The hazard is confabulation stabilization.

Stabilization occurs when:

- a story is repeated until it feels factual
- provenance is forgotten
- and contradiction becomes intolerable

Then the actor cannot repair.

The actor can only defend.

---

## Failure modes

### Retrospective certainty

The past becomes clean and obvious.

### Memory-as-weapon

Old stories are used to punish.

### Selective forgetting

Evidence that threatens identity disappears.

### External memory avoidance

Refusing records to preserve narrative freedom.

---

## Minimal implementation

A correction-capable actor preserves:

1. external memory for high-impact commitments and conflicts
2. provenance labeling for beliefs (“how do I know?”)
3. willingness to revise remembered narratives
4. refusal to use memory as punishment

If memory becomes certainty, correction collapses.
