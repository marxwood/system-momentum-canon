# Editorial Systems â€” ADRs (Derivative-Scoped)

This directory contains **derivative-scoped ADRs** for `derivatives/editorial-systems`.

These ADRs:

- apply only within this derivative,
- are non-canonical,
- MUST remain Canon-compatible,
- MUST NOT introduce new canonical primitives.

Authority level: **4** (see `governance/authority.md`).

---

## Relationship to System-level ADRs

- System-level ADRs live in `/adrs` (when present).
- Derivative-scoped ADRs live here.

If an ADR affects multiple derivatives, it does NOT belong here.

---

## Required References

Each ADR MUST include:

- Canon reference(s),
- Derivative scope statement (this derivative),
- Rejected alternatives,
- Non-goals.
