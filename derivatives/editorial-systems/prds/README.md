This directory contains **derivative-scoped PRDs** for `derivatives/editorial-systems`.

These PRDs:

- define normative correctness conditions for this derivative,
- are non-canonical,
- MUST remain Canon-compatible,
- MUST be implementation-agnostic.

Authority level: **4** (see `governance/authority.md`).

---

## Relationship to System-level PRDs

- System-level PRDs live in `/prds` (when present).
- Derivative-scoped PRDs live here.

If a PRD is intended to apply across multiple derivatives, it does NOT belong here.

---

## Required References

Each PRD MUST include:

- Canon reference(s),
- Derivative scope statement (this derivative),
- Requirements + Non-requirements,
- Validation approach (observable, not implementational).
