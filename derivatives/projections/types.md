# Projection Types

This document defines **canonical projection types**
as structural classifications derived from the Brain.

Projection types define **structural intent and scope**.
They do not define authority, validity, or usage behavior.

---

## Canonical Classification

All projections belong to exactly one canonical type:

- **Public Projection**
- **Operational Projection**
- **Analytical Projection**

This classification is **closed**.

---

## Type Definitions

- **Public Projection**  
  Projection intended for external visibility.

- **Operational Projection**  
  Projection intended for system operation and coordination.

- **Analytical Projection**  
  Projection intended for inspection and analysis.

---

## OSS Subtype

**OSS (Operational System Surface)** is a subtype of **Operational Projection**.

OSS is classified structurally as operational and system-wide.

---

## Non-Equivalence Rules

- Public ≠ Operational
- Operational ≠ Analytical
- Analytical ≠ Governance
- OSS ≠ Authority

Cross-type substitution constitutes a structural violation.
